# GOAL-013 Step 2.5 Development Report (2026-03-13)

## EN
### Scope
- Reworked the market-news presentation so it is embedded in the stock page flow instead of a sticky floating banner, and added an expanded reading dialog for dense translated headlines.
- Fixed stale overseas RSS pollution and the AI retry gap that could leave pending stock/sector/market facts permanently unprocessed.

### Development
- Backend freshness fixes:
  - `RssMarketNewsParser` now drops feed items older than 30 days and no longer treats missing/invalid publish times as “now”.
  - `LocalFactIngestionService` now appends a timestamp query parameter when requesting overseas RSS feeds to reduce stale upstream cache hits.
  - Domestic market facts built from the Sina roll feed now also discard stale entries beyond the 30-day window.
- Backend AI retry coverage:
  - `EnsureMarketFreshAsync` now runs `ProcessMarketPendingAsync` even when market data is already considered fresh if any `IsAiProcessed == false` market rows remain.
  - `EnsureFreshAsync` now does the same for stock and sector rows via `ProcessSymbolPendingAsync`, fixing the retry gap after earlier LLM failures.
- Frontend UI changes:
  - Replaced the sticky `terminal-market-banner` with an embedded `market-news-panel` that sits in the normal page flow.
  - Added an “展开阅读” action that opens a modal dialog showing the full market-news list for dense reading.
  - Kept the existing translated title, original-title fallback, sentiment, target, and AI tag rendering intact.
- Tests updated:
  - Added RSS recency coverage in `RssMarketNewsParserTests`.
  - Added stale-filter and pending-retry coverage in `LocalFactIngestionServiceTests`.
  - Added modal interaction coverage in `StockInfoTab.spec.js`.
  - Updated the Edge check selectors for the embedded market-news panel.

## ZH
### 范围
- 把大盘资讯从 sticky 悬浮条带改成股票页内嵌面板，并增加“展开阅读”弹窗，便于集中查看密集翻译资讯。
- 修复外媒 RSS 旧闻污染，以及“待补标本地事实在已有新鲜数据时被早退逻辑跳过”的 AI 重试缺口。

### 开发
- 后端时效修复：
  - `RssMarketNewsParser` 现在会直接丢弃超过 30 天的 RSS 项，并且不再把缺失/非法发布时间误当成“当前时间”。
  - `LocalFactIngestionService` 请求海外 RSS 时会附带时间戳 query 参数，尽量规避上游静态缓存污染。
  - 新浪滚动构造的大盘事实也加入了同样的 30 天时效过滤。
- 后端 AI 重试覆盖：
  - `EnsureMarketFreshAsync` 即使判定“大盘数据已新鲜”，只要仍有 `IsAiProcessed == false` 的 market 行，也会补跑 `ProcessMarketPendingAsync`。
  - `EnsureFreshAsync` 对 stock/sector 也做了同样处理，修复了此前 LLM 失败后可能永远得不到下一轮补标的问题。
- 前端 UI 改动：
  - 去掉 sticky 的 `terminal-market-banner`，改为页面流内的 `market-news-panel`。
  - 新增“展开阅读”按钮，可呼出弹窗查看完整大盘资讯列表。
  - 保留了译题、原题回退、情绪、靶点和 AI 标签展示。
- 更新测试：
  - `RssMarketNewsParserTests` 新增 RSS 时效过滤覆盖。
  - `LocalFactIngestionServiceTests` 新增旧闻过滤与待补标重试覆盖。
  - `StockInfoTab.spec.js` 新增大盘资讯弹窗交互覆盖。
  - Edge 脚本同步切换到新的内嵌面板选择器。
