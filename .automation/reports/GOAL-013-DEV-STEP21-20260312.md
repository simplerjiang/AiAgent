# GOAL-013 Step 2.1 Development Report (EN + ZH)

## EN
### Summary
Completed Step 2.1 rework for GOAL-013. Fixed LLM HTML/non-JSON fallback handling, removed local-news hard truncation, added sentiment labels, repaired sector/market local-fact refresh behavior, and made the intraday ticker clickable.

### Fixed Defects
- Added defensive HTML/non-JSON detection in `OpenAiProvider` and `StockAgentJsonParser` so gateway pages no longer crash stock-agent JSON parsing.
- Added `Sentiment` to local fact DTOs and classified local-news titles with deterministic keyword rules.
- Removed `.slice(0, 3)` and `.slice(0, 5)` truncation in `StockInfoTab.vue`; enabled scrollable local-news and ticker containers.
- Added clickable ticker items with `window.open(..., '_blank', 'noopener,noreferrer')` and pointer hover styling.
- Fixed `LocalFactIngestionService` refresh contention by separating market and per-symbol refresh locks.
- Expanded sector matching and market fallback logic in local fact ingestion so `level=sector` and `level=market` return stable data.
- Decoupled local-news bucket rendering from `/api/stocks/news/impact`, so local facts render even when impact analysis is still loading or fails.

## ZH
### 摘要
已完成 GOAL-013 的 Step 2.1 返工。修复了 LLM 返回 HTML/非 JSON 时的容错问题，去掉了本地新闻硬截断，增加了情感标签，修复了板块/大盘本地事实刷新与展示链路，并补齐了盘中消息带点击打开详情的交互。

### 修复内容
- 在 `OpenAiProvider` 与 `StockAgentJsonParser` 中增加 HTML/非 JSON 响应防御，避免网关页导致多 Agent JSON 解析崩溃。
- 为本地事实 DTO 增加 `Sentiment` 字段，并通过确定性关键词规则输出 `利好/中性/利空`。
- 删除 `StockInfoTab.vue` 中对本地新闻和消息带的 `.slice(0, 3)` / `.slice(0, 5)` 硬截断，改为可滚动容器。
- 为盘中消息带增加 `window.open(..., '_blank', 'noopener,noreferrer')` 点击打开详情能力，并补充手型/悬停态样式。
- 修复 `LocalFactIngestionService` 并发刷新竞争：拆分为“大盘一把锁 + 每个 symbol 一把锁”，避免 `/api/news` 与 `news/impact` 并发请求互相打架。
- 扩展板块关键词匹配与大盘回退逻辑，确保 `level=sector`、`level=market` 能稳定落库并返回数据。
- 将本地新闻卡片从 `/api/stocks/news/impact` 的成功条件中解耦，即使影响分析仍在加载或失败，本地事实卡片也必须显示。
