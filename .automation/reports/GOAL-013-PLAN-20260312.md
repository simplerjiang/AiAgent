- Execute Step 2 from `.automation/chatgpt_directives.md`: build the local-first fact hub for A-share stock news and sector/market context while keeping controlled internet search only for allowed non-domestic scenarios.

### Planned Work
- Add `LocalStockNews` and `LocalSectorReport` entities, register them in `AppDbContext`, and generate a migration.

  - Eastmoney announcement JSON API for stock announcements
  - Eastmoney company survey API for industry/sector metadata

### Validation Plan

2. Run frontend unit tests covering the stock page news integration.
3. Run `dotnet test` for the backend test project.
4. Verify DB schema with `sqlcmd` after migration/update.
5. If frontend changes remain, run `npm --prefix frontend run build`.
