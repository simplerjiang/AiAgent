# ISSUE-20260310-P3 Plan - White-box MCP/Skill + Unfinished Task Loop

## Goal
把 Agent 能力和任务执行流程从黑盒改为白盒：可见、可控、可回放、可恢复。

## Work Breakdown
1. Capability Registry
   - 记录 skill 名称、版本、输入输出契约、权限边界、执行副作用
2. 任务状态机
   - pending/running/succeeded/failed/cancelled
   - 失败重试次数与终止条件
3. 未完成任务定时执行
   - 扫描 pending/failed(retriable) 任务
   - 按优先级和租约机制执行，避免并发重复跑
4. 审计与可观测
   - 每步记录 traceId、duration、errorCode、retryCount
5. 最小可视化查询
   - 查看任务队列、失败原因、最近执行轨迹

## DoD
- 每个 skill 的调用都有权限与结果审计
- 未完成任务可自动续跑并受重试策略约束
- 状态转移与权限边界单测通过
