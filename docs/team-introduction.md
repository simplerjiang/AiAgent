## 我们能做什么

### 桌面软件开发

基于 .NET (WPF/WinForms/MAUI)、Electron 等主流框架，开发 Windows/macOS 跨平台桌面应用，适用于企业内部管理系统、专业工具软件、数据分析平台等场景。

### 网站与 Web 应用

前端精通 Vue、React 等现代框架，后端熟练运用 .NET、Node.js、Python，支持从企业官网、电商平台到复杂 SaaS 系统的全栈开发，兼顾性能与用户体验。

### 移动端 App 开发

覆盖 iOS 与 Android 双平台，支持原生开发（Swift/Kotlin）及跨平台方案（Flutter、React Native、.NET MAUI），满足社交通讯、金融工具、O2O 服务等多类型应用需求。

### 微信小程序 / 支付宝小程序

深度对接微信、支付宝生态，快速交付商城、预约、会员、营销裂变等小程序，帮助企业低成本触达用户。

### 数据与 AI 智能应用

具备大模型（LLM）集成、多智能体（Multi-Agent）系统、RAG 检索增强生成、数据爬取与分析等前沿 AI 工程能力，可为业务注入智能决策与自动化能力。

### 数据库与后端服务

精通 SQL Server、PostgreSQL、SQLite、Redis 等主流数据库，擅长高并发、高可用的后端架构设计与性能调优。

### UI/UX 设计服务

提供专业的用户界面与交互体验设计，涵盖产品原型、视觉设计、设计规范输出与前端还原度走查。擅长 B 端管理后台、数据可视化大屏、移动端 App 及小程序的 UI 设计，注重信息架构合理性与操作流畅度，确保产品"好看且好用"。

### DevOps 与云部署

熟悉 Docker 容器化、CI/CD 流水线、Azure/AWS/阿里云等云平台部署，确保系统稳定上线与持续迭代。
